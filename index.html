FloatingMenu demo

About FloatingMenu

DEV NOTE: not floating at the mo..

The FloatingMenu™ sits inline with the page, but when scrolling down we want to keep it in view! So as soon as it reaches the top it will get a fixed position. When scrolling back to the top it will take it's original position.

This page was created with a myrad of technologies, including basic HTML, LESS -using the SUIT CSS methodology (derived from BEMs) and React, written with the Babel transpiler. The whole is powered and packaged using Webpack.

Note that the links in the menu are fetched from the static html page. This wouldn't happen in a modern Single Page Application of course, but I think it shows nicely how modern tech such as React can be easily merged with legacy projects. Besides, I created this example in 4 hours, so I didn't have time to make it Über-fancy ☺︎
